# AI大模型导航

一个基于纯 HTML/CSS/JavaScript 构建的 AI 大模型聚合导航页，采用双轨道旋转交互设计，支持 PC 端和移动端自适应访问。

演示地址：https://aicollection.937788.xyz/

## 功能特点

- **双轨道交互设计**：外环展示主流 AI 大语言模型，内环展示 AI 视频/绘画创作工具
- **中心信息面板**：点击轨道上的任意图标，中心区域实时显示该工具的图标、名称、简介和直达链接
- **自动轮播**：外环每 4 秒自动切换高亮不同的 AI 模型
- **全屏模式**：支持一键全屏浏览
- **响应式布局**：PC 端横向排列，移动端自动适配为纵向布局


## 技术栈

- HTML5
- CSS3（Flexbox / Grid / 动画 / 响应式 Media Query）
- 原生 JavaScript（无框架依赖）


## 文件结构

```
index.html          主页面（单文件，所有样式和脚本内联）
/static/css/        外部样式文件（style.css / all.min.css）
/static/js/         外部脚本文件（main.js）
/static/picture/    工具图标资源
/js/                jQuery 库
/static/layer/      Layer 弹窗组件
```


## 使用说明

1. 直接浏览器打开 `index.html` 即可使用
2. 点击轨道上的任意图标，中心面板会显示该工具的详细信息
3. 点击「立即访问」按钮可跳转到对应官网
4. 支持全屏模式切换
